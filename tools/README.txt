TOOLS - Manager ServerPal
=========================

1) tool_dong_goi_du_an.bat
   - Dong goi toan bo du an.
   - Tu dong goi theo luong:
       build_manager_serverpal_exe.bat
       build_manager_serverpal_app_exe.bat
       Inno Setup 6 (ISCC.exe) de tao installer
   - Tao 1 file thanh pham de gui user:
       release\Manager_ServerPal_Setup_Latest.exe
   - Ban theo version van nam trong:
       release\Manager_ServerPal_Setup_v*.exe

2) tool_auto_gui_du_an_len_github.bat
   - Tu dong gui project len GitHub.
   - Thuc hien:
       git add -A
       git commit -m "..."
       git push origin <current-branch>
   - Neu khong co thay doi moi, tool se push branch hien tai.

Luu y:
- Can cai Git for Windows va da dang nhap quyen push.
- Nen kiem tra file nhay cam truoc khi push len repo public.
